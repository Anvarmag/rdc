# Шрифты

Локальная папка для хранения веб-шрифтов.

## Содержимое
В данный момент папка содержит файлы гарнитуры (woff/woff2) и используется как fall-back.

В рабочей версии сайта (v1.1.0) основной шрифт **Inter** загружается из **Google Fonts (CDN)** для максимальной производительности глобального кэширования браузерами:
`<link href="https://fonts.googleapis.com/...&family=Inter:...&display=swap" rel="stylesheet">`

Для версии сайта для слабовидящих (`accessible.html`) предпочтительно использование системных шрифтов (Arial/sans-serif), так как они обладают высокой читаемостью без необходимости скачивания кастомных начертаний.
