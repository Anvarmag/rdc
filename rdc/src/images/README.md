# Изображения

Папка для хранения растровых и векторных графических материалов проекта.

## Текущее состояние
В данный момент проект (v1.1.0) в значительной степени опирается на внешние заглушки-картинки с сервиса `picsum.photos` для демонстрационных целей. Единственный локальный ресурс — это логотип.

- **`logo-rdc.png`** — Основной логотип (Raster) Радиологического дата-центра. Используется в шапке (header) всех HTML документов.

## Для будущих разработчиков

1. При замене `picsum` заглушек на реальные фотографии врачей, помещайте изображения в эту папку (например: `images/team/doctor-1.jpg`).
2. **Форматы:** Старайтесь использовать конвертацию в `.webp` или `.avif` для больших фото, чтобы не терять прирост производительности статического сайта.
3. Обязательно сохраняйте соотношение сторон фотографий специалистов (рекомендуется `1:1` или `4:5` кроп) для ровной сетки на сайте.
