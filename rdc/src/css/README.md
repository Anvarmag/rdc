# Стили (CSS)

Эта папка содержит архитектуру самописных стилей проекта, которые не перекрываются стандартными utility-классами Tailwind.

## Структура файлов

- **`variables.css`** — CSS-переменные проекта. Содержит токены цветов, базовые размеры шрифтов и сетки. Вся цветовая палитра строится на этих константах, ссылаясь на цвета Tailwind, где это возможно.
- **`base.css`** — Базовые стили и CSS reset, импорт шрифтов (Inter) и глобальные настройки `body` (например, `scroll-behavior: smooth`).
- **`components.css`** — Набор стилей для специфичных, визуально сложных компонентов, функционал которых громоздко описывать одними лишь Tailwind-классами.
- **`utilities.css`** — Собственные утилиты, такие как `glass-effect` (размытие фона) или `gradient-text`.

## Как использовать
Все эти файлы подключены в `<head>` файлов `index.html` и `accessible.html`. БОльшая часть верстки страницы завязана на **Tailwind CDN**. Используйте эти файлы **только** тогда, когда эффекта невозможно добиться стандартными утилитами `tw-` (например `backdrop-filter: blur()`).
