# Журнал изменений

Все заметные изменения в этом проекте будут документироваться в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Семантического версионирования](https://semver.org/lang/ru/).

## [1.2.0] - 2026-02-27

### Исправлено
- **HTML-структура:** Исправлена вложенность секций — `how-it-works` не была закрыта, из-за чего форма заявки и «Преимущества» оказывались внутри неё.
- **Переключатель языков (build.py):** Исправлен порядок операций — regex для switcher теперь выполняется до замены `data-i18n`, что предотвращает перезапись текста кнопок.
- **Cookie-баннер:** Убрана битая ссылка `/privacy-policy`, добавлены `data-i18n` атрибуты для перевода.
- **Копирайт:** Добавлен `id="currentYear"` для динамической подстановки года через `initFooter()`.
- **Кнопка «Показать врачей»:** Заменена бессмысленная ссылка `#team` на кнопку с вызовом `TeamModule.showAll()`.

### Добавлено
- Переводы формы заявки, карты и cookie-баннера (7 ключей × 3 языка).
- `sitemap.xml` обновлён с hreflang и языковыми версиями.

### Удалено
- Устаревший параметр `?v=20260118` из подключения `translations.js`.
- Ссылки на несуществующие `pages/*.html` из `sitemap.xml`.
- Ссылка на `node_modules` из `robots.txt`.

---

## [1.1.0] - 2026-02-26

### Изменения в архитектуре и Оптимизация
- **Отказ от клиентского JS для переводов:** модуль `i18n.js` удален.
- Внедрен статический генератор на Python (`build.py`) для мультиязычности.
- Добавлен английский язык (EN).
- Языковые версии (RU, EN, TT) теперь существуют в виде независимых HTML-файлов (например, `index-en.html`).
- Навигация переключения языков переведена на обычные ссылки (`<a href>`) вместо кнопок с `preventDefault()`.
- Восстановлены и зафиксированы атрибуты `data-i18n` в исходных файлах `index.html` и `accessible.html`.

## [1.0.0] - 2024-01-18

### Добавлено

#### Структура проекта
- Профессиональная файловая структура репозитория
- Модульная CSS-архитектура (`css/variables.css`, `css/base.css`, `css/components.css`, `css/utilities.css`)
- Модульная JavaScript-архитектура с отдельными модулями
- Данные вынесены в отдельные файлы (`js/data/doctors.js`, `js/data/services.js`)

#### Страницы
- Главная страница (`index.html`) с полным лендингом
- Версия для слабовидящих (`accessible.html`)
- Страница 404 (`404.html`)

#### Функциональность
- Адаптивный хедер с мобильным меню
- Динамический рендеринг карточек врачей
- Модальные окна профилей специалистов
- Форма обратной связи
- Анимации и эффекты (glass, float, gradient)
- Оптимизация для мобильных устройств

#### SEO и документация
- Полные мета-теги для SEO
- Open Graph теги для социальных сетей
- `robots.txt` для поисковых роботов
- `sitemap.xml` карта сайта
- Полная документация `README.md`
- Руководство для разработчиков `CONTRIBUTING.md`
- Политика безопасности `SECURITY.md`
- Лицензия MIT

#### Оптимизация
- Lazy loading для изображений
- Content-visibility для секций
- Поддержка prefers-reduced-motion
- Оптимизация для touch-устройств

### Технологии
- HTML5 с семантической разметкой
- Tailwind CSS (CDN)
- Vanilla JavaScript (ES6+)
- Lucide Icons
- Google Fonts (Inter)
