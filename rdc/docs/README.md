# Документация проекта РДЦ

Добро пожаловать в папку с документацией разработчика и ИИ.
Здесь собраны служебные Markdown-файлы, описывающие архитектуру, историю и правила работы с кодовой базой.

## Содержимое папки

- **`ARCHITECTURE.md`** — Подробное техническое описание стэка проекта (отказ от сборщиков, использование Vanilla JS + Tailwind CDN, принцип работы `build.py`). Это самый важный файл для понимания того, как работает генерация сайта.
- **`LOCALIZATION.md`** — Руководство по редактированию словаря (`translations.js`) и работе с утилитой генератора для сборки языковых версий сайта.
- **`CHANGELOG.md`** — История версий (включая переход на статический движок в v1.1.0).
- **`CONTRIBUTING.md`** — Руководство по развертыванию среды для новых участников и соглашения о стилях.
- **`SECURITY.md`** — Политика безопасности и контактная информация.
- **`AUDIT.md`** — Архивная копия результатов первого масштабного аудита кодовой базы (до v1.1.0).

> **AI Context:** Сводные правила для ИИ-ассистентов (вроде Cursor или Copilot) вынесены в корень проекта, в файл `.cursorrules`. Главная витрина проекта (`README.md`) также расположена в корне репозитория.
